# emoji-unicode
Для виведення смайлів, емоджі в консолі потрібно перевести її в юнікод. Після цього появляється можливіть виводити смайти, емоджі за допомогою простого `"System.out.println()"` в якому можна вказати код смайла або вставити в форматі юнікод.

Найти коди смайлів можна за допомогою сайтів такий як цей: https://unicode-table.com/ru/sets/emoji/
